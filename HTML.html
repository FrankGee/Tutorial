Tutorial: Intro to HyperText Markup Language(HTML)

What is HTML?

  1. HTML is the standard markup language to create web pages.
  2. First introduced in 1982 and gets several updates, now is HTML5
  3. Now could contain almost anything you want, images, videos, and etc...
  4. Maintained by Wolrd Wide Web Consortium(W3C).

What's important

HTML is constructed by html elements which consist of tags enclosed in brackets

  • HTML files are stored in .html documents, very few might in .htm(DOS)
  • Always include a <!DOCTYPE html> tag at the beginning of your html file.
  • Most of the tags come in pairs, <html> and </html>, <body> and </body>.
  • Some tags come alone like <img>.

HTML Basic Syntax

  1. Headings 
    <h1>this is heading one</h1>
<h2>this is heading two</h2>

    HTML headings specify the font size of characters. h1 indicates the biggest heading.

    Result: 
    this is heading one
    

    this is heading two

  2. Paragraphs 
    <p>this is a paragraph.</p>
<p>this is a paragraph.</p>

    <p> is used to construct paragraphs.
    Result: 
    this is a paragraph.
    

    this is a paragraph.

  3. link 
    <a href="http://www.w3schools.com">this is a link</a>

    <a> tags are used to refer links.

    Try it!
