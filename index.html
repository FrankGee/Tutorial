Online Tutorial of Web Developing: A Pre-look of CSE-3901

Welcome to our Web Tutorial!

Since the commercialization of the web, web development has been a growing industry. The growth of this industry is being pushed especially by businesses wishing to sell products and services to online customers. -Wikipedia

As you are entering the course of CSE-3901, this online tutorial provides you with sets of basic concepts of web development. This tutorial covers the area from the backside logic programming and frontside design. By starting with this tutorial, you will be able to get into the course very soon.

The topics we will cover include:

  • HTML
  • CSS
  • JavaScript
  • Ruby

What we recommend is that you can start up with the tutorial of HTML, in which you can see how a webpage is composed in code. After that you can go on with CSS which makes easily changes in the HTML Style. Then you can go on with the JavaScript Part. At last you can go further on the Ruby part which makes the webpage into a real web app.
